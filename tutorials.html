How to Start a Slip Knot
  1. Make a loop with the yarn tail crossing the working yarn.
  2. Insert hook under the working yarn, pull it through the loop.
  3. Pull both ends to snug; loop should slide on hook.

How to Crochet a Chain (CH)
  1. With slip knot on hook, yarn over (yo).
  2. Pull through loop on hook — 1 chain.
  3. Repeat; count “V” shapes.
▶ Play Chain Tutorial
Single Crochet (SC)
  1. Insert hook in stitch.
  2. Yo, pull up loop (2 loops).
  3. Yo, pull through both loops.
▶ Play Single Crochet Video
Half Double Crochet (HDC)
  1. Yo, insert hook in stitch.
  2. Yo, pull up loop (3 loops).
  3. Yo, pull through all 3.
▶ Play HDC Video
Double Crochet (DC)
  1. Yo, insert hook in stitch; yo, pull up loop (3 loops).
  2. Yo, pull through 2 (2 loops).
  3. Yo, pull through 2 to finish.
▶ Play DC Video
Magic Ring (MR) / Magic Circle
  1. Wrap yarn around fingers to form loop.
  2. Insert hook under loop, yo, pull up.
  3. Ch to secure; work sts in ring; pull tail to close.
▶ Play Magic Ring Video

Classic Granny Square
  1. Ch 4 join ring (or MR); ch 3 counts dc.
  2. Work 3‑dc clusters + ch‑spaces for corners.
  3. Join & add color rounds as desired.
